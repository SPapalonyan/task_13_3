# Анализ тональности отзывов

В рамках данного проекта выполнен анализ тональности текстовых отзывов с использованием предобученной модели обработки естественного языка. В качестве модели использована модель RuBERT для анализа тональности (blanchefort/rubert-base-cased-sentiment), позволяющая классифицировать тексты на основе их эмоциональной окраски.

## Описание задачи

Целью работы является автоматическое определение тональности отзывов, представленных в CSV-файле. Для каждого текста необходимо определить, является ли он нейтральным, положительным или отрицательным, а также визуализировать общее распределение тональностей.

## Используемые технологии

В проекте применяются следующие библиотеки:
- pandas — для работы с табличными данными
- transformers — для загрузки и использования предобученной модели
- torch — для выполнения вычислений модели
- matplotlib — для визуализации результатов

## Этапы выполнения

1. Установка и импорт необходимых библиотек.
2. Загрузка датасета с отзывами в DataFrame.
3. Инициализация токенизатора и модели.
4. Реализация функции анализа тональности текста.
5. Применение функции ко всем отзывам.
6. Сохранение результатов в новый столбец predicted_sentiment.
7. Подсчет количества отзывов по каждой категории.
8. Построение столбчатой диаграммы распределения тональностей.

## Результат

В результате выполнения работы:
- каждому отзыву присвоена тональность (Neutral, Positive, Negative);
- получена наглядная визуализация распределения отзывов по тональности;
- сформирован DataFrame с дополнительным аналитическим признаком.

## Вывод

Использование предобученных моделей позволяет эффективно решать задачи анализа тональности без необходимости обучения собственной модели. Полученные результаты могут быть использованы для анализа клиентского опыта, улучшения качества сервиса и принятия управленческих решений.
